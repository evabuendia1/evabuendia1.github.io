Domain Name System

Le DNS (Domain Name System) peut être expliqué comme le système essentiel d'Internet qui est utilisé comme traducteur qui change les noms de domaine lisibles par les humains en adresses IP utilisées par les ordinateurs pour communiquer. C’est donc le moyen de convertir le langage habituel en langage numérique.

Votre navigateur ne supporte pas la lecture des vidéos.

Bienvenue dans le Chapitre sur Internet

Dans ce chapitre, nous allons comprendre comment fonctionne l’internet et son histoire, les programmes derrière la communication entre les différents réseaux, les protocoles de sécurité et les différents langages employés avec leur correspondante traduction. Pour cela, je présente les sujets suivants: l’histoire d' internet, le Protocole TCP, le Protocole IP, DNS, le Routage, les Réseaux pair à pair et la Conversion décimal à binaire.
